# private-furniture-6305 ( Clone-BoatLifeStyle.com )

A fully functional clone of the **boAt Lifestyle** website, developed as a part of a collaborative project. The website offers a seamless shopping experience, with features like browsing products, adding them to the cart, and completing purchases. This project highlights the use of **local storage** for data management, offering users a smooth and efficient interface.

---

## **Overview**

boAt Lifestyle is your go-to digital shopping assistant, dedicated to saving you time and money while ensuring top-quality products. The clone replicates the core functionalities and user interface of the original website. Here’s what you can expect:

### **Features:**

- A **homepage** featuring top categories and products.
- **Product detail pages** showcasing images, detailed descriptions, and pricing.
- A **shopping cart** to manage selected items.
- A streamlined **checkout process** for order completion.
- Fully responsive design, optimized for all devices.

### **Tech Stack:**

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Local Storage for data handling
- **Deployment**: Hosted on **Netlify**

## **Getting Started**

Follow these steps to run the project on your local machine:

1. **Clone the Repository**

```bash
  git clone https://github.com/deepakpatil26/BoatLifeStyle-clone
```

2. **Navigate to the Project Directory**

```bash
   cd BoatLifeStyle-clone
```

3. **Install Dependencies**

```bash
   npm install
```

4. **Start the Development Server**

```bash
   npm start
```

5. **Open in Browser**
   Visit: http://localhost:3000
